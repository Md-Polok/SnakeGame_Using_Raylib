# 🐍 Snake Game (C++ & Raylib)



A classic, fully playable Snake game built from scratch using **C++** and the **Raylib** library. This project was created to practice object-oriented programming, game loops, collision detection, and game state management.

## ✨ Features
* **Classic Gameplay with a Twist:** Eat food to grow your tail and increase your score, but watch out for the randomly placed stone obstacles!
* **Dynamic Difficulty:** The snake's movement speed progressively increases as you eat more food, making the game more challenging over time.
* **Sound Effects:** Features custom audio for eating food, hitting milestones (every 100 points), and game-over collisions.
* **Game States:** Includes a proper Start Screen and seamless restart loops.

## 🎮 Controls
